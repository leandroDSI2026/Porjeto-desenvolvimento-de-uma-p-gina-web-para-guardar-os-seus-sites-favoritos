# 📍 Meus Sites Favoritos: Interior do Brasil

Este projeto é uma galeria interativa que destaca destinos turísticos icônicos localizados no interior do Brasil. Ele foi desenvolvido como parte de um estudo prático de desenvolvimento web front-end.

## 📖 Sobre a Atividade
O objetivo principal foi construir uma página web funcional que integrasse:
- **Semântica HTML:** Uso correto de tags como `<h1>`, `<a>` e `<img>`.
- **Estilização CSS:** Layout limpo e organizado para melhor experiência do usuário.
- **Interatividade:** Links que direcionam para os portais oficiais de cada destino.

## 🗺️ Destinos Listados
1. **Bonito (MS)** - Famoso pelas águas cristalinas e ecoturismo.
2. **Ouro Preto (MG)** - Cidade histórica com arquitetura colonial preservada.
3. **Jalapão (TO)** - Destino de aventura com dunas e fervedouros.
4. **Chapada dos Veadeiros (GO)** - Berço de cachoeiras e natureza exuberante.
5. **Capitólio (MG)** - Conhecido pelos seus imensos canyons e represas.

## 🛠️ Tecnologias
- **HTML5**: Estrutura da página.
- **CSS3**: Design e responsividade básica.
- **Git/GitHub**: Controle de versão e hospedagem.
